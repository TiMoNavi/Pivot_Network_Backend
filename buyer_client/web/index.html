Pivot Buyer Console

这个本地网页只和本地 buyer 进程通信，再由本地 buyer 进程与平台后端交换必要数据。页面本身不直接碰 Docker、WireGuard、远端主机或 seller 节点。

代码与工作区执行

支持单文件代码、本地 zip/目录、GitHub 仓库。页面只把必要输入交给本地 buyer 进程，再由本地 buyer 进程与平台后端交互。

Archive / GitHub
