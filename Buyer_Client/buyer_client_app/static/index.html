Pivot Network

买家本地客户端

买家购买的不是卖家真实 IP，而是一段由平台按 Docker Swarm 风格编排出来的 runtime bundle 会话。

当前推荐流程

  1. 登录 buyer 账号
  2. 浏览算力 Offer
  3. 创建 Order
  4. 激活订单并申请 runtime bundle
  5. 使用 gateway / overlay / WireGuard 材料进入 runtime

正式语义

  • Offer = 可售卖的调度模板
  • Order = 一次租用请求
  • AccessGrant = 会话期接入材料
  • Runtime Bundle = runtime service + gateway service + session network

这个页面当前是本地骨架，核心流程通过 `/local-api/*` 提供。
